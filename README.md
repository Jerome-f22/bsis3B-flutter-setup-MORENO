Flutter Quiz App
Overview

This is a simple Quiz App built with Flutter. It lets users answer multiple-choice questions, see their score in real time, and get results at the end of the quiz. The app is designed to be clean, interactive, and easy to use.

What the App Does:
  - Shows a start screen with a “Start Quiz” button
  - Displays one question at a time
  - Lets users select one answer per question
  - Highlights answers as correct (green) or incorrect (red)
  - Tracks the score while playing
  - Shows a final results screen with:
      Total score
      Percentage
      Pass/fail feedback
  - Allows users to restart the quiz anytime

Features:
  - Simple and clean design
  - Easy-to-understand quiz flow
  - Instant feedback on answers
  - Works on Android, iOS, and other Flutter-supported platforms
